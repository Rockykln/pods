# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

Short version: be civil, focus on the technical question, no harassment.
Bad-faith behaviour in issues, PRs, or any other project space gets you
removed.

## Reporting

Email **[contact@rockykln.com](mailto:contact@rockykln.com)**. Reports
are handled confidentially.
